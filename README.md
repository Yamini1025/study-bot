# Study Bot
Study Bot is an application that is designed to help students study more effectively using AI. Users can paste their notes as text, and the app will generate summaries, explanations, and practice quiz questions.  
This project is being developed collaboratively by two CS sophomores at San Jos√© State University as a learning-focused, frontend-heavy project integrating AI through APIs.

## Features
- **Text Input**
  - Users can paste class notes or study material as text

- **AI-Powered Actions**
  - **Summarize:** Generate a concise summary of the input notes
  - **Explain:** Explain concepts in simpler terms for better understanding
  - **Quiz:** Generate either 5 or 10 practice questions based on the notes
  
## Tech Stack
- **Frontend**
  - HTML/CSS  
  - JavaScript  
  - React
